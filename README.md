# ⭐ Fundamental Booster Project

Hey Users 👋  
Welcome to the **Fundamental Booster Project**!

This is a beginner-friendly Python project created using Jupyter Notebook. It helps you understand basic programming concepts like variables, input, and output in a simple way.

---

## 📌 Project Description

This project collects user information such as:

- 🧑 Name  
- 🎂 Age  
- 📏 Height  
- 📅 Birth Year  
- 🏙️ City (optional)

After taking input, it displays the information in a clean and formatted way.

---

## 📍 Features

✔️ Takes user input  
✔️ Uses variables and data types  
✔️ Performs basic calculations (like birth year)  
✔️ Displays formatted output  
✔️ Easy to understand for beginners  

---

## 🛠️ Tools Used

- 🐍 Python  
- 📓 Jupyter Notebook  

---

## ▶️ How to Run

1. Open the `.ipynb` file in Jupyter Notebook  
2. Run all cells step by step  
3. Enter your details when asked  
4. View the output 🎉
